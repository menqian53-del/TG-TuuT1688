# L2TP VPN 一键安装脚本

## 下载安装

在 Linux 服务器上执行以下命令：

```bash
curl -O https://raw.githubusercontent.com/menqian53-del/TG-TuuT1688/main/l2tp_install.sh && chmod +x l2tp_install.sh && bash l2tp_install.sh
```

## 支持系统

- CentOS 6+
- Debian 7+
- Ubuntu 12+

## 使用说明

安装完成后，可以使用以下命令管理用户：

```bash
l2tp -a    # 添加用户
l2tp -d    # 删除用户
l2tp -l    # 列出所有用户
l2tp -m    # 修改用户密码
```
